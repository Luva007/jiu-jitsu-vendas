COMO EDITAR

1. Abra a pasta no VS Code.
2. Edite o index.html para trocar textos e link do checkout.
3. No botão de compra, procure por:
   https://SEU-LINK-DA-KACTO-AQUI.com
   e troque pelo seu link da Kacto.
4. Para trocar a imagem do fundo:
   abra style.css e procure por background dentro da classe .hero.
5. Para testar, abra index.html no navegador ou use Live Server no VS Code.
